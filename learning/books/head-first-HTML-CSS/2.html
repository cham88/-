深入了解超文本

主要内容
知识点 说明
<a>元素 a元素，也叫anchor（锚点）元素，既可以用来链接到外部地址实现页面跳转功能，也可以链接到当前页面的某部分实现内部导航功能。
加入链接，用法<a href="链接地址">链接</a>
href属性为Hypertext Reference缩写，告诉浏览器链接的目标文件
相对路径 相对是相对于当前位置，以当前HTML文档的位置为起点开始寻径
绝对路径 从根目录到目标文件的路径
<img> 插入图像，用法<img src="图像地址" alt="当图像无法显示时，显示的文字">
img是内联元素
