块元素和内联元素 块元素：每个块元素都单独显示，前后都有换行
内联元素：不单独显示，会随波逐流，如a、img、q、em、strong等

void元素（空元素） 没有内容的元素，如<br/> ，<img>也是void元素
有序列表<ol>、无序列表<ul> 有序列表ol——ordered list
无序列表ul——unordered list
